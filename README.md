# 🍰 Fino Gesto — Página Inicial

Este projeto consiste no desenvolvimento da **página inicial de uma loja de doces finos**, criada com foco em **design elegante, responsividade e experiência do usuário**.

A interface foi construída utilizando **Bootstrap 5**, garantindo que o layout se adapte perfeitamente a diferentes tamanhos de tela, como celulares, tablets e desktops.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem **mobile-first**, garantindo:

* Boa visualização em dispositivos móveis
* Adaptação para tablets
* Layout organizado e centralizado em desktops

Componentes como **navbar, carousel e cards** foram utilizados de forma estratégica para manter a consistência visual em todas as resoluções.

---

## 🎯 Objetivo

Criar uma **landing page moderna e atrativa** para uma confeitaria de doces finos, destacando:

* Produtos em destaque
* Identidade visual delicada
* Facilidade de navegação

---

## 🧁 Funcionalidades da Página

* Menu de navegação responsivo
* Carrossel de imagens (banner principal)
* Seção de novidades
* Listagem de produtos em formato de cards
* Botão de “Mais Produtos”
* Rodapé com:

  * Informações de contato
  * Newsletter
  * Redes sociais

---

Este projeto representa **apenas a página inicial** da loja, podendo ser expandido futuramente com:

* Página de produtos detalhados
